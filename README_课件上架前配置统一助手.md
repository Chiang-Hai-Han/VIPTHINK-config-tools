# 课件上架前配置统一助手

作者：海瀚

这个版本把封面上传、关联线上作业、资源复制整合到一个本地助手和一张配置表。

## 启动

双击：

```text
run_prelaunch_assistant.bat
```

统一助手端口：

```text
http://127.0.0.1:8769
```

## 统一配置表

编辑：

```text
课件上架前配置表.xlsx
```

每行维护同一个课件的上架前配置：

```text
课件编码 | 课件名称 | 封面图片 | 作业课件编码 | 资源来源课件 | 封面上传状态 | 课件关联状态 | 资源复制状态 | 备注
```

如果不填：

- 作业课件编码默认是：`课件编码 + _hw`
- 资源来源课件默认把 `_TW` 替换成 `_YY`
- 封面图片默认从 `待上传图片文件夹` 里按课件编码查找图片

## 网页插件

1. 启动 `run_prelaunch_assistant.bat`。
2. 在浏览器扩展管理页重新加载 `chrome_extension`。
3. 在课件管理后台刷新或查询一次，让插件捕获登录状态。
4. 打开插件，点击 `课件上架前配置`。
5. 根据需要执行：
   - 检查封面上传任务
   - 开始上传封面
   - 检查线上作业关联
   - 开始关联线上作业
   - 资源复制检查
   - 开始资源复制

资源复制当前只复制：

```text
作业题目、学习报告
```

不会复制：

```text
课后总结
```

## 覆盖确认

正式写入前，如果网页上已经存在对应内容，插件会弹出确认：

```text
覆盖 | 全部覆盖 | 取消
```

- 覆盖：只覆盖当前课件，然后继续下一行。
- 全部覆盖：本次循环后续都自动覆盖。
- 取消：中断本次流程。

## 结果文件

统一结果会写入：

```text
课件上架前配置结果.json
```

同时状态会回写到：

```text
课件上架前配置表.xlsx
```
