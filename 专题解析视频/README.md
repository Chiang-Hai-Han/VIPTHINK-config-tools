# 专题解析视频

这个目录是独立的新功能，不影响现有项目。

当前已落 4 个能力：

1. 生成配置表  
2. 监听你手动操作一次时的接口请求  
3. 根据字符映射，自动把“来源题目名称”补到配置表里
4. 独立插件批量执行“下载来源视频 -> 上传到目标题目”

## 配置表说明

配置表文件名：`专题解析视频配置表.xlsx`

字段：

- `目标题目名称`
- `来源题目名称`
- `目标课件编码`
- `来源课件编码`
- `题号`
- `指导视频复制状态`
- `备注`

默认会放一条例子：

- `目标题目名称`：`s1_v7_01-04_TW第1题`
- `来源题目名称`：`s1_v8_01-04_YY第1题`

## 推荐使用顺序

### 1. 先生成配置表

双击：

- `run_create_topic_video_sheet.bat`

### 2. 如果要批量补“来源题目名称”

先检查映射文件：

- `字符映射.json`

当前默认内置了这组映射：

- `s1_v7_01-04_TW` -> `s1_v8_01-04_YY`

然后双击：

- `run_fill_source_titles.bat`

### 3. 监听你手动操作一次

双击：

- `run_listen_topic_video_requests.bat`

脚本会打开一个可调试浏览器。你登录后台后，手动做一遍：

- 从来源题目复制指导视频
- 上传到目标题目

当前默认理解为：

- 下载 `s1_v8_01-04_YY第1题` 的指导视频
- 上传到 `s1_v7_01-04_TW第1题`

监听结果会保存到：

- `监听结果/专题解析视频接口监听_时间戳.json`

### 4. 分析监听结果

双击：

- `run_analyze_topic_video_capture.bat`

它会把监听文件里和“题目名称 / 视频 / 上传 / 复制”相关的字段路径尽量找出来，方便下一步把真正的批量上传脚本补齐。

### 5. 真正执行批量复制

先双击：

- `run_topic_video_assistant.bat`

然后在 Chrome 扩展管理页加载这个目录：

- `chrome_extension_topic_video`

再去后台页面刷新一次，让插件捕获登录态。

最后打开插件，按这个顺序：

1. `检查本地助手`
2. `空跑检查`
3. `开始批量复制`

执行结果会回写到配置表的：

- `指导视频复制状态`
- `备注`

## 现阶段说明

现在已经不是监听阶段了，当前脚本会按配置表逐行执行：

1. 搜索来源题目  
2. 读取来源题目指导视频  
3. 下载视频并重新上传 OSS  
4. 登记附件  
5. 写回目标题目  
6. 回写配置表状态
